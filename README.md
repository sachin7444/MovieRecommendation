# 🎬 Movie Recommender System

A simple and interactive **Movie Recommender System** built using **Python** and **Streamlit** that suggests similar movies based on a selected title.

---

## 🚀 Features

-  Select any movie from the dropdown
-  Get top recommended similar movies
-  Displays movie posters
-  Fast and lightweight Streamlit app
-  Content-based filtering

---

## 🛠️ Tech Stack

- **Frontend/UI:** Streamlit  
- **Backend:** Python  
- **Libraries:**
  - pandas
  - numpy
  - scikit-learn
  - requests
  - pickle

---

## ⚙️ How It Works

1. Movie dataset is preprocessed  
2. Features like genres, cast, keywords are combined  
3. Converted into vectors using CountVectorizer  
4. Cosine similarity is computed  
5. On selection, similar movies are recommended
